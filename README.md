# Marketing Hooks

**Puzzle-Driven Content Model for Marketing**

An AI skill for creating compelling marketing hooks and content structures based on proven psychological frameworks.

## 🎯 What This Skill Does

Helps you create attention-grabbing marketing content using the **Puzzle-Driven Model** - a framework that leverages how the human brain responds to questions and extreme cases.

**Perfect for:**
- Social media content creators
- Marketing professionals
- Newsletter writers
- Content strategists
- Anyone who needs to capture attention

## 🧠 Core Framework

### Four Question Types

1. **How-Type** (Method) - "How to 10x your productivity with AI?"
2. **Why-Type** (Reason) - "Why 90% of AI tools fail?"
3. **What-Type** (Definition) - "What is Product-Market Fit?"
4. **WoW-Type** (Extreme Case) ⭐ - "Gen-Z makes $100K/month with AI"

### Content Tree Structure

```
Question (Root)
    ↓
Viewpoint (Trunk)
    ↓
Reasoning (Branches)
    ↓
Evidence (Leaves)
```

**Quality check:** A good piece of content = a tree with lush branches and leaves.

## 💡 Usage Examples

### Example 1: Generate a WoW-Type Hook

**Prompt:**
```
Create a WoW-type hook for my AI productivity tutorial
```

**Output:**
```
"I replaced my entire team with one AI tool - here's how"

Why it works:
- Extreme efficiency claim (1 tool vs. team)
- Creates puzzle: "What tool can do this?"
- Triggers curiosity gap
```

### Example 2: Analyze Existing Content

**Prompt:**
```
Why does this title work well? "Gen-Z makes $100K/month with AI"
```

**Output:**
```
This is a WoW-type hook that works because:

1. Extreme contrast:
   - Young age (Gen-Z) + High income ($100K/month)
   - Low probability combination

2. Creates puzzle:
   - "How is this possible?"
   - Brain wants to fill in the missing logic

3. Tree structure potential:
   - Root: How can someone so young earn so much?
   - Trunk: AI democratizes income
   - Branches: Specific tools and strategies
   - Leaves: Real examples and proof
```

### Example 3: Create Platform-Specific Content

**Prompt:**
```
Write a Xiaohongshu video script about AI productivity tools using the Puzzle-Driven Model
```

**Output:**
```
[Opening - 5-10s]
"You know what? I used one AI tool to finish a month's work in 3 days"

[Pain Point - 10-15s]
"Are you also working overtime every night, never finishing your work?"

[Solution - 120-180s]
"Today I'll teach you 3 techniques:
- Technique 1: [specific method + demo]
- Technique 2: [specific method + demo]
- Technique 3: [specific method + demo]"

[Results - 20-30s]
"Look, now I only need 2 hours a day to finish what used to take all day"

[CTA - 5-10s]
"Follow me for more AI efficiency tools"
```

### Example 4: Amplify an Existing Hook

**Prompt:**
```
Make this hook more engaging: "AI tools improve efficiency"
```

**Output:**
```
Original: "AI tools improve efficiency"
❌ Too generic, no puzzle, no engagement

Amplified versions:

1. Add extreme numbers:
   "5 AI tools that 10x your efficiency"

2. Add time pressure:
   "10x your efficiency in 30 days with these AI tools"

3. Add contrast:
   "One AI tool vs. entire productivity system"

4. Add failure narrative:
   "Why 90% fail with AI tools (and how to succeed)"

Recommended: Option 4 (Why-type with failure narrative)
- Creates strongest puzzle
- Relatable problem
- Implies insider knowledge
```

### Example 5: Create Newsletter Subject Lines

**Prompt:**
```
Generate 5 newsletter subject lines about AI automation using different question types
```

**Output:**
```
1. WoW-Type:
   "I automated my entire workflow with ChatGPT - $50K saved"

3. How-Type:
```

## 🎓 Key Concepts

### WoW-Type Content (Highest Engagement)

**Formula:** Create probability-defying extreme cases

- **Discover extremes** → Get traffic
- **Become extreme** → Build brand

**Why it works:** The brain automatically fills in missing logic, creating engagement stickiness.

**Examples:**
- Age + Income: "Gen-Z makes $100K/month"
- Time + Scale: "10K followers in 3 days"
- Replacement: "One tool replaced my entire team"

### Gan Method Content Model

**Structure:**
1. **Core viewpoint** (Trunk) - Your main argument
2. **Sub-arguments** (Branches) - Supporting points
3. **Evidence** (Leaves) - Must be:
   - Unexpected
   - Memorable
   - Vivid

**Quality test:** Can you visualize a tree with lush branches and leaves?

## 📊 Success Metrics

Track these to validate your hooks:

**Xiaohongshu:**
- Click-through rate (CTR) on cover
- Average watch time
- Save rate

**X (Twitter):**
- Engagement rate (likes + retweets + replies)
- Thread completion rate
- Profile visits from post

**Newsletter:**
- Open rate (subject line effectiveness)
- Read time
- Click-through on CTAs

**Benchmark targets:**
- WoW-type: 2-3x average engagement
- How-type: 1.5-2x average engagement
- Why-type: 1.5-2x average engagement
- What-type: 1x average engagement (educational baseline)
